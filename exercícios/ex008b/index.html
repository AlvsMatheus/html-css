Código / Pré formatação

O código ''code'' e ''pre'' + ''code'' serve para representar códigos de maneira mais visível

o código ''code'' deixa todos os termos de mesmo tamanho de espaço

isso facilita para visualizar códigos, como document.getElementBy.. 

  
o ''Pre''', junto com o código ''code'', ajuda a 
quebrar as linhas, isso ajuda principalmente em códigos
atalho parra quebrar espaço: ctrl tab - quebrar / tab - adicionar
  
        

        
Citações Simples

        
Para uma citação simpes usamos o código q que significa quote, ou seja, citação

    é preferíel, nas citações, utilizar
    o comando q (quote), ao invés de usar aspas

Citações completas (em bloco)

utilizamos o código blockquote

caso quisermos uma citação em um bloco de espaço diferente
obs: Podemos usar o ''cite'' logo após o blockquote, dentro do mesmo código, e copiar a url do site citado (não mostra a url, apenas facilita a busca )
