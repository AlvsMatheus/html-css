Negrito / Destaque

Como vimos, há diferenças entre um código morfológico e semântico, já que, o código semântico é focado na funcionalidade, enquanto o código morfológico é focado no design

Eles podem ser visualmente parecidos, mas, em questão de linguagem, no html, é preferível o uso do código semantico, até porque, o foco em design é do css
